# Security Policy

## Supported scope

This repository contains:

- a Codex skill,
- validation scripts,
- install logic for copying the skill into a local Codex skills directory.

## Reporting a vulnerability

Please do not open a public issue for security-sensitive problems.

Instead, contact the maintainer privately with:

- affected file or component,
- impact,
- reproduction steps,
- suggested mitigation if available.

## Good reports include

- exact script or file path,
- platform details,
- whether the issue affects installation, local file handling, or validation behavior,
- whether user data or repository contents could be exposed or modified unexpectedly.

## Response goal

Reasonable effort will be made to confirm, prioritize, and fix valid reports.
